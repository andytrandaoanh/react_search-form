This standard project has been created and tested by Andy Tran.
The npm_modules has been removed to facilitate easy copy.
To start, copy the entire folder and rename it to your taste.

1. Edit the package.json to give the project a new name:
[package.json]
"name": "Your New Project Name",

2. Restore the npm_modules:
[command line] npm install

3. Build the source code:
[command line] npm run build

4. Start the project:
[command line] npm start

NOTES

The project include the following main packages:
- ReactJS
- Bootstrap 
- Jquery
- Babel
- Webpack with minimal optimization 

and many other modules.
